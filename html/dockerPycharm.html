    Connect Docker with Pycharm to run Python Code







                

    Before starting make sure the following prerequisites are met :-

    1. Docker is installed and is running.
    2. PyCharm Pro and Git is installed

    a) Create a Git Repository with your desired name.
    b) Create Docker file with no extension and add these lines of code.

    FROM python:3
    ADD my_script.py /
    RUN pip install pystrich
    CMD [ "python", "./my_script.py" ]

    c) Now, create my_script.py file and these lines of code.

    from pystrich.datamatrix import DataMatrixEncoder
    encoder = DataMatrixEncoder('This is a DataMatrix.')
    encoder.save('./datamatrix_test.png')
    print(encoder.get_ascii())

    d) The repository should be look like in the image shown in which my_script.py file will be there in src folder. Open PyCharm and clone your repository. The screenshot is shown here.




    e) Open File > Settings and check docker is running as shown in image.

    f) Go to add configuration which is there in top right of PyCharm.

    g) Add the file name and click ok.

    h) Run the program. The program will run in the container as shown in the screenshot.
